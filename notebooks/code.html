average_speed.calc_g_prime(Nj, Q_cj, rho_j, rho_a, g=9.81)¶

This function calculates the reduced gravity $g’$. Returns reduced gravity matrix numpy.ndarray(Ny,Nx).

Parameters:
  • Nj (int) – Number of sediment layers used in simulation.
  • Q_cj (numpy.ndarray((Ny,Nx,Nj))) – Concentration of j’th sediment layer. Fraction, i.e. Q_cj :math in [0,1]
  • rho_j (numpy.ndarray((Nj))) – $rho_j[j] =$ Density of j’th sediment layer
  • rho_a (float) – Density of ambient fluid
  • g (float) – Gravitational acceleration.

>>> Nx=Ny=3

average_speed.calc_potEnergy(Q_th, g_prime, rho_c, A)¶

This fuction calculates and returns the potential energy of the turbidity current for cells in an (Nx by Ny) grid. Each cell has an area A, height Q_th, and some density rho_c.

Parameters:
  • Q_th (numpy.ndarray(Ny,Nx)) – Turbidity current thickness. Unit = m
  • g_prime (numpy.ndarray(Ny,Nx)) – Reduced gravity for all cells. Unit = m/s^2
  • rho_c (numpy.ndarray(Ny,Nx)) – Entry [i,j] is the current density in cell [i,j]. Unit = kg/m^3
  • A (float) – Area of hexagonal grid cell. Unit = m^2
average_speed.calc_rho_c(Nj, Q_cj, rho_j, rho_a)¶

This function calculates the current density rho_c, for all Ny x Nx cells. If a cell has no sediment current, the current density is just the ambient current. Otherwise, the current density is a weighted average of the densities present in the cell. The weights are the volume concentration of each sediment.

Parameters:
  • Nj (int) – Number of sediment layers used in simulation.
  • Q_cj (numpy.ndarray((Ny,Nx,Nj))) – Concentration of j’th sediment layer. Unit = 1. Fraction, i.e. Q_cj :math in [0,1]
  • rho_j (numpy.ndarray((Nj))) – $rho_j[j] =$ Density of j’th sediment layer. Unit = kg/m^3
  • rho_a (float) – Density of ambient fluid. Unit = kg/m^3

Examples:

>>> import numpy as np
>>> Nj =2
>>> Nx=Ny=3
>>> Q_cj = np.zeros((Ny,Nx,Nj))
>>> Q_cj[1,1,0] = 1 # 100 % concentration of sediment no. 0.
>>> Q_cj[1,0,1] = 1 # 100 % concentration of sediment no. 1.
>>> Q_cj[1,2,:] = 0.5 # 50/50 mix of sediment no. 0 and 1.
>>> Q_cj[0,0,:] = 1/3 # Equal mix between two sediments and ambient density.
>>> rho_j = np.array([10,5])
>>> rho_a = 3
>>> calc_rho_c(Nj,Q_cj,rho_j,rho_a)
... array([[  6. ,   3. ,   3. ],
...        [  5. ,  10. ,   7.5],
...        [  3. ,   3. ,   3. ]])
